# Rush

A secure, scalable cloud storage solution built with Vinext, Cloudflare Workers, D1, R2, and KV.

## Project Structure

- `src/app`: App router (React Server Components)
- `src/schema.sql`: D1 database schema
- `wrangler.toml`: Cloudflare configuration

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to Cloudflare
npm run deploy
```
